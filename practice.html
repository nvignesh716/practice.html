I.Introductory Terms

1.HTML

HTML is a language that browsers use
in order to interpret webpages.
It stands for Hypertext MarkUp Language

2.Components of Web

Client-It includes your computer and Browser
Internet &
Servers
The Client & Servers interact through
a process called Hypertext Transfer Protocol(HTTP)

3.Elements

Contents enclosed within a tag is called Elements

4.Void Tags

A Tag with no closing Tag is called as a Void Tag
The image tag is one of a void tag

II.Advanced Terms

1.White Space

It refers to text on multiple lines to form a paragraph
The br Tag and Paragraph Tag a.k.a. p Tag are used for the purpose.

2.Container Elements

Elements that do nothing but contains stuff inside Elements.
Few Examples are-
Span element and
Div Elements

3.Block Elements vs.Inline Elements

Elements which creates a rectangular block around its cotents
which has its own length & breadth are called as Block Elements
Few Examples of Tags that creates Block Elements are as follows-
1.Paragraph Tag(a.k.a. p-Tag)
2.div Tag
On the other hand,
Inline Elements are those elements which does not create
a rectangular block around its elements.
Few Examples are as follows-
1.Image Tag(a.k.a.img-tag)
2.Strong Tag

III.HTML Attributes

HTML Attributes are tags which hava an Attribute(a.k.a.ATTR)
and each Attribute equals to a value(i.e.URL).
